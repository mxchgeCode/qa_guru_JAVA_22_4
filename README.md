Задание
```
1. Есть ли разница между $("h1 div"); и $("h1").$("div"); - может ли привести к тому что, поиск найдёт разные элементы?
```
Разница есть, например:
```
<body>
<h1>test 1</h1>
<h1>
<div>test 2</div>
</h1>
</body>
```
```
$("h1 div"); найдет div.
$("h1").$("div"); упадет с ошибкой, так как будет искать div внутри первого найденного h1.
```
